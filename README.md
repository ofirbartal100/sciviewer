# SciViewer

## Description
Overrides the plt.show() function to solve the X11 forwording problem. and mimic natural behavior.

## Features
- SciViewer Main Image
- SciViewer Thumbnails

## Usage
1. After installation, open a Python file.
2. Use the command palette (`Ctrl+Shift+P`) to run `My Extension: Start`.
3. The extension will perform the task.
